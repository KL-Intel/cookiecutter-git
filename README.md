# cookiecutter-docs-md
Cookiecutter for **Markdown Documentation**!

## Features
 - Markdown documentation
 - Simple project structure
 - Apache license v2

## Upcoming
 - License customization
 - Gitignore customization
 - Gitlab support

## Requirements
 - [python][1]
 - [cookiecutter][2]

## Usage
    $ cookiecutter gh:webevllc/cookiecutter-aws-django
